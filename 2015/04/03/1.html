GitHub Pagesでサイト作ってみる（Middlemanで）インストール編。

GitHub Pages Middleman

Posted on2015-04-03

まずはMiddlemanのインストール。
Mac OSX YosemiteのRubyのバージョン。2.0.0になってる。たしかMavericksは1.9.3だったような。
だいたいMiddlemanのインストールページの手順でやれば大丈夫なはず。

Middleman: インストール

ruby -v
ruby 2.0.0p481 (2014-05-08 revision 45883) [universal.x86_64-darwin14]

初期状態でのgem（RubyGems）のバージョンはわかりません。

gem -v
2.4.6

以下のようにすればいい。

gem install middleman

permissionのエラーが出たら以下のようにすればいい。

sudo gem install middleman

インストールが完了したら任意の名前でプロジェクトを作成する。
例として「my_example」として作る。

middleman init my_example
cd my_example
middleman server

ポート4567でローカルサーバーが起動する。
http://localhost:4567/

== The Middleman is loading
== LiveReload is waiting for a browser to connect
== The Middleman is standing watch at http://0.0.0.0:4567
== Inspect your site configuration at http://0.0.0.0:4567/__middleman/

これでとりあえずはMiddlemanのインストールは完了。
